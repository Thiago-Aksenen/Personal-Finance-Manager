Este trabalho apresenta o desenvolvimento de um sistema web criado para facilitar o
cuidado com a vida financeira do usuário, oferecendo um espaço simples e organizado para
registrar receitas, despesas e acompanhar o saldo atualizado.

![1764161161356576](https://github.com/user-attachments/assets/d36784f9-9998-47de-968a-191f75846f9e)

A proposta surge da necessidade cada vez maior de ter um apoio confiável no controle do dinheiro, reduzindo
preocupações e tornando o processo de organização financeira mais claro e acessível. O
sistema foi desenvolvido utilizando o framework Laravel, usando Bootstrap para layout e
arquitetura baseada no padrão MVC, o que garante uma estrutura eficiente e fácil.
